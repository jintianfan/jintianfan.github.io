jvm关闭

关闭方式

JVM的关闭意味着将停止系统中所有的任务，可以由其自动关闭也可以主动触发。下面的几个条件都可以使JVM关闭：

  1. 所有的非守护都已经运行结束
  2. 调用了System.exit方法
  3. 杀死JVM进程 (比如SIGKILL或者Runtime.getRuntime().halt 、断电)
  4. 通过系统平台发送关闭信号（比如按Ctrl+C或者SIGINT信号）

使用后关闭钩子的前提是JVM正常退出，上述第三种场景强制杀死JVM不会触发钩子函数执行。

Runtime.getRuntime().addShutdownHook(new Thread());
